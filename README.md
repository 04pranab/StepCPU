# StepCPU

A step-by-step implementation of a minimal 4-bit CPU in Python.

## Features

- 8-register file (x0 hardwired to 0)
- 4-bit ALU with overflow behavior
- Instruction decoding (R-type, I-type)
- Branching (BEQ, BNE, BLT)
- Data memory (LOAD / STORE)
- Loop execution (array sum demo)

## Progression

1. Registers
2. ALU
3. CPU Core
4. Branching
5. Memory + Programs

## Example

Program to sum an array:

`mem = [3,5,2,4,1] → result = 15`


## Goal

Understanding computer architecture from first principles.
